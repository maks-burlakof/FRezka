# Installation

Create a venv, install requirements.

Create `.env` file in directory `app/`. Fill the following info:

```text
DATABASE_HOSTNAME=
DATABASE_PORT=
DATABASE_PASSWORD=
DATABASE_NAME=
DATABASE_USERNAME=
SECRET_KEY=
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=525600
REZKA_URL="http://hdrezka.ag/"
```
